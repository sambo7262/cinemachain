```python
async def _maybe_advance_session(tmdb_id: int, db: AsyncSession) -> None:
    result = await db.execute(
        select(GameSession).where(GameSession.status == "active")
        .options(selectinload(GameSession.steps))  # avoid lazy="raise"
    )
    session = result.scalar_one_or_none()
    if session and session.current_movie_tmdb_id == tmdb_id:
        session.status = "awaiting_continue"
        await db.commit()
        logger.info("Session %d advanced to awaiting_continue for tmdb_id=%d", session.id, tmdb_id)
```

Note: paused sessions have status="paused" — the where clause `status == "active"` correctly excludes them.

    Create frontend/nginx.conf:
    ```nginx
    server {
        listen 80;
        server_name _;
        root /usr/share/nginx/html;
        index index.html;

        # SPA routing — REQUIRED for React Router client-side routes
        # Without this, refreshing /game/123 returns nginx 404
        location / {
            try_files $uri $uri/ /index.html;
        }

        # API proxy to backend (backend service name in docker-compose)
        # Trailing slash on proxy_pass strips the /api prefix:
        # GET /api/game/sessions/active → forwarded as GET /game/sessions/active
        location /api/ {
            proxy_pass http://backend:8000/;
            proxy_set_header Host $host;
            proxy_set_header X-Real-IP $remote_addr;
            proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        }

        # Gzip
        gzip on;
        gzip_types text/plain text/css application/json application/javascript text/xml application/xml;
    }
    ```

    Note on API proxy: the nginx /api/ location strips the /api prefix when proxying to backend:8000 (trailing slash on both sides). This is correct — FastAPI routes are /game/sessions/*, not /api/game/sessions/*.
