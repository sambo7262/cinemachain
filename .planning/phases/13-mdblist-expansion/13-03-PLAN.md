   ```typescript
   interface RatingsData {
   ```

   Badge definitions array — each entry has: key (matching RatingsData field), label, icon/element, formatter:
   - `imdb_rating` — icon: `<SiImdb size={14} />`, format: `(v) => v.toFixed(1)` (e.g. "7.8")
   - `rt_score` — icon: emoji or styled text, format: `(v) => v + "%"` (e.g. "94%")
   - `rt_audience_score` — icon: emoji or styled text, format: `(v) => v + "%"` (e.g. "87%")
   - `metacritic_score` — icon: styled text "MC", format: `(v) => v + "%"` (e.g. "81%")
   - `mdb_avg_score` — icon: styled text "MDB", format: `(v) => v.toFixed(1)` (e.g. "7.4")
   - `vote_average` — icon: `<SiThemoviedatabase size={14} />`, format: `(v) => v.toFixed(1)` (e.g. "7.6")
   - `letterboxd_score` — icon: `<SiLetterboxd size={14} />`, format: `(v) => v.toFixed(1)` (e.g. "3.9")

   Variant subsets:
   ```typescript
   const VARIANT_KEYS: Record<BadgeVariant, string[]> = {
     card: ["imdb_rating", "rt_score", "rt_audience_score", "metacritic_score"],
     splash: ["imdb_rating", "rt_score", "rt_audience_score", "metacritic_score", "mdb_avg_score", "vote_average", "letterboxd_score"],
     tile: ["imdb_rating", "rt_score"],
   }
   ```

   Render logic:
   - Filter badges to variant's keys
   - For each badge: skip if value is null, undefined, or 0 (sentinel)
   - Render flex-wrap container: `<div className="flex flex-wrap gap-1.5 items-center">`
   - Each badge: small inline-flex items-center gap with icon + formatted score text
   - Use `text-xs` sizing for compact display
   - Icon colors: use each icon's brand color where available (SiImdb default color is fine), use neutral for text labels

   For RT icons specifically, use:
   - Tomatometer: text with tomato emoji prefix or a styled red text "RT"
   - Audience: text with popcorn emoji prefix or a styled text "Aud"

   The component should be clean and self-contained. Export as named export `RatingsBadge`.
