## Decisions Made
- useLoadingMessages returns `string | null` (not empty string) so JSX `{message && ...}` guard is clean
- `showMoviesSpinner` state and its `useEffect` removed entirely — the hook replaces the 1s delay guard since the message only shows when fetching is true
- Direct TMDB ID input placed outside the `u.suggestions.length === 0` conditional so it appears for both no-results and low-confidence suggestion rows
- `current_movie_watched=False` explicit in import even though default is False — documents intent clearly for future readers
