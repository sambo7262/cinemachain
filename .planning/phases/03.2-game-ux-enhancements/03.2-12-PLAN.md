**FAIL:** Zero eligible actors returned (blank list with no names at all).

**Test 2 — Regression 2: No stale actor flash on actor change**

1. Using the same session (or a fresh one), mark the current movie as watched.
2. Open Eligible Actors — click one actor (e.g. Actor A). The Eligible Movies tab opens and loads Actor A's movies.
3. Click the Back button to return to Session Home.
4. Click Continue the chain again (or navigate back to Eligible Actors tab without advancing).
5. Click a DIFFERENT actor (Actor B).

**PASS:** The Eligible Movies tab immediately shows a loading state (spinner if >1s) or Actor B's movies directly. No flash of Actor A's movie list appears.
**FAIL:** Actor A's movies are visible for more than a brief moment before Actor B's movies appear.

---

**Test 3 — Gap 3: Eligible Movies tab loads without actor selected**

**PASS:** The tab immediately begins loading (spinner appears within 1 second if fetch is slow) and then shows a list of eligible movies spanning all eligible actors. Filters (genre, runtime, MPAA rating) work on this list. The tab does NOT show "Pick an actor from the Eligible Actors tab to see movies."
**FAIL:** Tab shows empty state or "Pick an actor..." message with no data load triggered.
