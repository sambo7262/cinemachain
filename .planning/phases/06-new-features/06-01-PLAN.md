4. Create `backend/app/services/settings_service.py`:
   ```python
   from cryptography.fernet import Fernet, InvalidToken
   from sqlalchemy import select
   from sqlalchemy.ext.asyncio import AsyncSession
   from app.models import AppSettings
   from app.settings import settings as env_settings
   ```

   Functions:
   - `_get_fernet() -> Fernet | None`: Returns Fernet instance if `env_settings.settings_encryption_key` is non-empty, else None.
   - `encrypt_value(plaintext: str) -> str`: If fernet available, encrypt; else return plaintext.
   - `decrypt_value(ciphertext: str) -> str`: If fernet available, decrypt (with InvalidToken fallback returning raw value); else return ciphertext.
   - `async def get_all_settings(db: AsyncSession) -> dict[str, str | None]`: SELECT all from app_settings, decrypt values where `is_secret=True`, return as `{key: value}` dict.
   - `async def get_setting(db: AsyncSession, key: str) -> str | None`: Get single setting by key, decrypt if secret.
   - `async def save_settings(db: AsyncSession, settings_dict: dict[str, str | None]) -> None`: For each key/value pair, upsert into app_settings. Keys containing "key", "token", or "password" are `is_secret=True` and values are encrypted before storage. Use `merge()` pattern for upserts.
   - `async def is_tmdb_configured(db: AsyncSession) -> bool`: Check if `tmdb_api_key` exists and is non-empty in app_settings.
   - `async def migrate_env_to_db(db: AsyncSession) -> bool`: If app_settings table is empty, read values from `env_settings` (the pydantic Settings instance) and write them to DB using `save_settings`. Known keys to migrate: `tmdb_api_key`, `tmdb_base_url` (hardcode default `https://api.themoviedb.org/3`), `radarr_url`, `radarr_api_key`, `radarr_quality_profile`, `sonarr_url`, `sonarr_api_key`, `plex_token`, `plex_url`, `tmdb_cache_time`, `tmdb_cache_top_n`. Return True if migration occurred.

   from fastapi import APIRouter, Depends

   Router prefix: `/settings`, tags: `["settings"]`

   Pydantic models:
   - `SettingsResponse(BaseModel)`: All setting keys as `str | None` fields: `tmdb_api_key`, `tmdb_base_url`, `radarr_url`, `radarr_api_key`, `radarr_quality_profile`, `sonarr_url`, `sonarr_api_key`, `plex_token`, `plex_url`, `tmdb_cache_time`, `tmdb_cache_top_n`
   - `SettingsUpdateRequest(BaseModel)`: Same fields, all optional
   - `SettingsStatusResponse(BaseModel)`: `tmdb_configured: bool`, `migrated_from_env: bool = False`

   Endpoints:

   - `GET /settings/status` -> `SettingsStatusResponse`: Calls `is_tmdb_configured(db)`
