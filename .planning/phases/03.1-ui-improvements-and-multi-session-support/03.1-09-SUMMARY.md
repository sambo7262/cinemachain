## Files Created/Modified

## Decisions Made
- `getActiveSession` kept in api.ts — it may be used by NavBar or other components; removing it was not required
- `key={defaultTab}` on Tabs chosen over controlled tab state — forces remount which reliably resets to new default without additional state management complexity
- `pauseMutation`/`resumeMutation`/`endMutation` removed entirely (not just UI hidden) — dead code elimination, API functions (`pauseSession`, `resumeSession`, `endSession`) left in api.ts since removing them was not required per plan
