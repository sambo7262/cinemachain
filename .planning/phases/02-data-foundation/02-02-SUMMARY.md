## Decisions Made

- lazy="raise" on all relationships: async SQLAlchemy cannot lazy-load without an active event loop; all callers must use explicit eager loading (selectinload/joinedload)
- WatchEvent.movie_id nullable: a Plex scrobble may arrive before TMDB data is fetched; the event should be stored immediately and linked to Movie later
- Migration hand-authored rather than autogenerated: no live PostgreSQL available at plan execution time; the migration precisely mirrors the ORM model definitions
