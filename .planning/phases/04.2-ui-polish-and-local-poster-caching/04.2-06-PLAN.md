  <action>
In `compose.yaml`, find the `backend` service `volumes` section which currently contains:
```yaml
    volumes:
      - /volume1/docker/appdata/cinemachain/backend:/app/data
```

Add the static posters volume as a second entry:
```yaml
    volumes:
      - /volume1/docker/appdata/cinemachain/backend:/app/data
      - /volume1/docker/appdata/cinemachain/static:/app/static
```

This bind-mount maps the NAS host path `/volume1/docker/appdata/cinemachain/static` to the container's `/app/static` directory. This is the directory that:
1. FastAPI's `StaticFiles` mount serves from (`app.mount("/static", StaticFiles(directory="static"), ...)`)
2. The nightly poster download pass writes to (`static/posters/{tmdb_id}.jpg`)
3. The frontend accesses via nginx proxy at `/api/static/posters/{tmdb_id}.jpg`

No other services need this volume — add to backend only.
