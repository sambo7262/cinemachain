# Phase 4.2 Plan 05: Frontend Feature Wiring Summary

**One-liner:** SessionCounters expanded to 5 stats; GameSession mounts PosterWall, handles chain dead-ends, and adds random-pick dialog; GameLobby session cards show compact stats line.

## What Was Built

### Task 1: SessionCounters — 5-stat expansion (TDD GREEN)

Rewrote `frontend/src/components/SessionCounters.tsx` completely:

- Extended `SessionCountersProps` with `stepCount: number`, `uniqueActorCount: number`, `createdAt: string`
- Added `formatSessionAge(createdAt)` helper producing human-readable ages: `5m ago`, `2h 15m ago`, `3d ago`
- Added Steps, Actors, and Started stat blocks with separator dividers
- All `SessionCounters.test.tsx` tests turned GREEN (were RED from 04.2-01 TDD setup)

### Task 2: GameSession — PosterWall + random pick + dead-end + new props

Six targeted changes to `frontend/src/pages/GameSession.tsx`:

1. **PosterWall mount**: `<PosterWall posters={posterWallData} />` as first child of root div. Root div uses `cn("min-h-screen flex flex-col", posterWallData.length < 5 ? "bg-background" : "")` so background falls through to PosterWall's dark overlay when 5+ posters are available.
2. **posterWall query**: `useQuery<PosterWallItem[]>` with `staleTime: 5 * 60 * 1000` — posters change infrequently.
3. **Dead-end detection**: `isDeadEnd = isWatched && !eligibleActorsFetching && eligibleActors.length === 0 && hasIneligibleActors`. Dead-end state replaces entire actors tab content with "Chain dead end" heading, descriptive text, Delete Last Step (outline) and End Session (destructive) buttons.
4. **Random Pick**: `handleRandomPick` picks a random index from `filteredMovies` (not `allEligibleMovies` — respects active filters). Random button added to the sort/filter controls row. Random Pick Dialog shows poster, title, year/runtime, with Keep Browsing and Request This Movie buttons.
5. **SessionCounters new props**: Call site updated to pass `stepCount`, `uniqueActorCount`, `createdAt`.
6. **useNavigate added** for End Session navigation to `/`.

### Task 3: GameLobby — session card stats line

Added to `frontend/src/pages/GameLobby.tsx`:

- `formatSessionAge` helper (same implementation as SessionCounters — no shared util)
- Stats line: `{session.watched_count} watched · {session.step_count ?? 0} steps · started {formatSessionAge(session.created_at ?? "")}` rendered below existing card content
