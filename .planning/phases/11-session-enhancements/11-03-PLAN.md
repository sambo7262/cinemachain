  <what-built>
    Complete Save and Shortlist features for Game Mode eligible movies:
    - Backend: 2 new DB tables (session_saves, session_shortlist), 7 new API endpoints, eligible-movies response includes saved/shortlisted flags, request_movie auto-clears shortlist and removes picked-movie save
    - Frontend: Star/ListCheck icons on movie row posters, save star in splash dialog, filter toggle buttons (Saved / Shortlist / Clear Shortlist), amber/blue row tints, filter stacking, shortlist 6-item cap, filter-aware empty states
  </what-built>
  <how-to-verify>
    **Pre-requisite:** Run Alembic migration on NAS Docker: `docker exec cinemachain-backend alembic upgrade head`

    **1. Save feature (SESS-01, SESS-02):**
    - Open an active game session with eligible movies visible
    - Click the star icon (bottom-left of any movie poster) — star should fill gold, row should get subtle amber tint
    - Click the star again — star should return to outline, amber tint removed
    - Save 2-3 movies, then refresh the page — saved movies should still show filled gold stars (persistence)

    **2. Saved filter (SESS-03):**
    - Click "Saved ★" toggle button in filter bar — list should show only saved movies
    - Click "Saved ★" again — list returns to all movies
    - Combine with other filters (e.g., genre or MPAA) — filters should stack

    **3. Shortlist feature (SESS-04):**
    - Click the shortlist icon (bottom-right of any movie poster, looks like a checklist) — icon should fill blue, row should get subtle blue tint
    - Add 6 movies to shortlist — the shortlist icon on remaining un-shortlisted movies should become disabled (faded)
    - Click "Shortlist ✓" toggle button — list should show only shortlisted movies
    - Click "Clear Shortlist" button — all shortlist items should clear, filter should show all movies again

    **4. Shortlist auto-clear (SESS-04 lifecycle):**
    - Add 2-3 movies to shortlist
    - Pick one of them (click a selectable movie, confirm in splash dialog "Add to Session")
    - After the movie is added to the chain, shortlist should be empty (verify by checking if shortlist icons are all outline)

    **5. Splash dialog save star (SESS-01):**
    - Click any movie to open the splash dialog
    - Verify a star icon appears in the dialog header next to the movie title
    - Click the star to save/unsave — star should toggle between filled gold and outline

    **6. Both saved + shortlisted (visual):**
    - Save a movie AND shortlist it — row should show both amber and blue tints overlapping
