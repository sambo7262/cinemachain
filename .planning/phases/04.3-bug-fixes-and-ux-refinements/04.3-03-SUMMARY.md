## Decisions Made

- `sortCol` default stays `"rating"` with `sortDir` `"desc"` to preserve existing UI behavior
- New column sort direction defaults: `"desc"` for rating, `"asc"` for year/runtime/mpaa (natural reading direction)
- `accumulatedMovies` reset effect depends on `[selectedActor?.tmdb_id, sortCol, sortDir, allMovies, debouncedSearch]` — any filter or sort change resets the list to page 1
- Load More hidden when `debouncedSearch` is truthy — backend returns all matching results without pagination in search mode
- `prevAccumulatedCountRef` tracks pre-append count to attach `firstNewResultRef` to the correct boundary row for smooth scroll
