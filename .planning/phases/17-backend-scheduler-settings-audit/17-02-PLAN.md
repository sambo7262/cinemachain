**Part A: Add mdblist_nightly_job to mdblist.py**

Add a new async function `mdblist_nightly_job` in `backend/app/routers/mdblist.py`. This is a scheduled variant of `_run_backfill` with these differences:

1. **Re-fetch threshold filter:** Query movies WHERE `mdblist_fetched_at IS NULL OR mdblist_fetched_at < cutoff` where `cutoff = datetime.utcnow() - timedelta(days=refetch_days)`. Read `refetch_days` from DB setting `mdblist_refetch_days` (default "90" if not set).

2. **Priority ordering:** `nulls_first(asc(Movie.mdblist_fetched_at))` then `Movie.vote_count.desc().nulls_last()`. Never-fetched movies first, then oldest-fetched, both sub-ordered by popularity.

3. **Quota respect:** At the top of the function, check remaining quota. If `_state.calls_used_today >= _state.daily_limit`, log "MDBList nightly: quota exhausted, skipping" and return immediately. Initialize quota from DB at start (same pattern as `start_backfill`).

4. **Uses `_bg_session_factory`** for DB access (same as `_run_backfill`).

5. **Reuses `_increment_quota`** for each API call.

6. **Rate limit:** `await asyncio.sleep(1.0)` between calls (1 req/s, matching existing `_run_backfill` pattern).

7. **Logging:** Log start, total eligible movies, progress every 500 movies, and completion with count.

```python
async def mdblist_nightly_job():
    """Scheduled MDBList backfill: fills never-fetched movies, refreshes stale ones."""
    logger.info("mdblist_nightly_job: starting")

    async with _bg_session_factory() as db:
        # Init quota from DB
        calls_str = await settings_service.get_setting(db, "mdblist_calls_today")
        reset_str = await settings_service.get_setting(db, "mdblist_calls_reset_date")
        _state.calls_used_today = int(calls_str) if calls_str else 0
        _state.daily_limit = 25000
        today_str = datetime.utcnow().strftime("%Y-%m-%d")
        if reset_str != today_str:
            _state.calls_used_today = 0

        if _state.calls_used_today >= _state.daily_limit:
            logger.info("mdblist_nightly_job: quota exhausted (%d/%d), skipping", _state.calls_used_today, _state.daily_limit)
            return

        # Read refetch threshold
        refetch_str = await settings_service.get_setting(db, "mdblist_refetch_days") or "90"
        refetch_days = int(refetch_str)
        cutoff = datetime.utcnow() - timedelta(days=refetch_days)

        result = await db.execute(
            select(Movie).where(
                or_(
                    Movie.mdblist_fetched_at.is_(None),
                    Movie.mdblist_fetched_at < cutoff,
                )
            ).order_by(
                nulls_first(asc(Movie.mdblist_fetched_at)),
                desc(Movie.vote_count).nulls_last(),
            )
        )
        movies = result.scalars().all()

    logger.info("mdblist_nightly_job: %d movies eligible for fetch/refresh", len(movies))
    # ... same per-movie loop as _run_backfill:
    # - get api_key from settings
    # - for each movie: check quota, call mdblist API, parse response, update movie, _increment_quota, sleep(1.0)
    # - log progress every 500
```

Import `timedelta` from `datetime`, `desc` from `sqlalchemy` if not already imported. The `or_`, `nulls_first`, `asc`, `select` are already imported in mdblist.py.
